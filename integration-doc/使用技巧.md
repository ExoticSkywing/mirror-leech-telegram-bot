💡 更多实用技巧：
1️⃣ 批量转移整个文件夹
# 转移整个文件夹（不是单个文件）
/clone cd2:/panwebdav/某个文件夹 -up p1
2️⃣ 转移到 Telegram 频道
# 把 WebDAV 的文件直接发到 TG 频道
/clone cd2:/panwebdav/文件.pdf -up p2
3️⃣ 使用键名简化命令
# 既然你已经配置了 p3 = cd2:/panwebdav
/clone p3/文件名 -up p1
# 或者转移整个 p3 到 p2
/clone p3 -up p2
4️⃣ 批量操作（Bulk）
创建一个文本文件，每行一个路径：
cd2:/panwebdav/book1.epub
cd2:/panwebdav/book2.epub
cd2:/panwebdav/book3.epub
回复这个文件：
/clone -b -up p1

✅ WebDAV → 115 云盘

✅ 任意路径互相复制
